# Netflix Data Analysis (Exploratory Data Analysis)

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix dataset to understand patterns in Netflix content such as movies, TV shows, genres, countries producing content, and growth of the platform over time.

The analysis uses **Python, Pandas, Matplotlib, and Seaborn** to clean the data and generate meaningful visual insights.

---

## Objectives

The main goals of this project are:

* Analyze the distribution of **Movies vs TV Shows**
* Identify **top countries producing Netflix content**
* Explore the **growth of Netflix content over time**
* Analyze **popular genres on Netflix**
* Study **content ratings and audience categories**
* Understand **movie duration patterns**

---

## Dataset

The dataset contains information about Netflix titles including:

* Title
* Type (Movie or TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genres (listed_in)
* Description

Dataset file used:
`netflix_titles.csv`

## Exploratory Data Analysis Performed

### 1. Dataset Overview

* Checked dataset structure
* Identified missing values
* Analyzed data types

### 2. Movies vs TV Shows Distribution

Analyzed how Netflix content is divided between movies and TV shows.

### 3. Content Added Over Time

Examined how Netflix's content library has grown over the years.

### 4. Top Countries Producing Content

Identified which countries produce the most Netflix titles.

### 5. Most Popular Genres

Analyzed the most common genres available on Netflix.

### 6. Content Rating Distribution

Explored the target audience categories for Netflix content.

### 7. Movie Duration Analysis

Studied the distribution of movie lengths.

---

## Key Insights

* Netflix contains **more movies than TV shows**.
* The majority of content is produced by the **United States**.
* Netflix content additions increased rapidly after **2015**.
* **Drama, Comedy, and Documentaries** are among the most common genres.
